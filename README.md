# ORBIT CAMPUS ( smart campus management system)

This is a comprehensive college management system designed to streamline various administrative and academic processes. It provides a centralized platform for students, teachers, parents, and administrators to manage their daily activities efficiently.

## Features

*   **User Authentication:** Secure login for students, teachers, parents, and administrators.
*   **Dashboards:** Personalized dashboards for each user role, providing quick access to relevant information.
*   **Student Management:** Manage student information, including academic records, attendance, and fees.
*   **Fee Management:** Track fee payments, generate fee receipts, and send reminders.
*   **Document Management:** Upload, organize, and share academic documents and resources.
*   **Academic Calendar:** Keep track of important academic events, holidays, and examinations.
*   **Notifications:** Real-time notifications for important updates and announcements.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

*   [Node.js](https://nodejs.org/) (v14 or later)
*   [npm](https://www.npmjs.com/)
*   A web server (like Apache or Nginx) with PHP support for the backend.
*   A MySQL database.

### Installation

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/your-username/school-management-system.git
    cd school-management-system
    ```

2.  **Install frontend dependencies:**

    ```bash
    cd frontend
    npm install
    ```

3.  **Set up the backend:**

    *   Move the `backend/php` folder to your web server's document root.
    *   Create a MySQL database and import the `database.sql` file.
    *   Configure the database connection in `backend/php/config/database.php`.

## Running the Project

1.  **Start the frontend development server:**

    ```bash
    cd frontend
    npm run dev
    ```

    This will start the Vite development server, and you can access the application at `http://localhost:5173`.

2.  **Ensure your backend server is running.**
